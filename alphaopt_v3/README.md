# AlphaOpt: Intelligent ETF Allocation & Risk Control System

An Investment Decision Engine Based on Portfolio Optimization.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run alphaopt_app.py
```

## Features

- 30 ETFs across multiple asset classes
- 3-year historical analysis (2022-2024)
- 3 risk-optimized portfolios
- Interactive visualizations
- Efficient frontier chart
- Asset class based allocation view

## Pages

- **Overview**: ETF universe and asset distribution
- **Analysis**: Performance metrics and rankings
- **Correlation**: Heatmap with value annotations and diversification insights
- **Portfolio**: Allocation recommendations with efficient frontier

## Data

All data files should be in the `data/` folder.
